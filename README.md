# Schoology to Google Classroom

## Instructions
Please download the .ipynb file, upload to google colab to start. Refer to the pdf with any questions and for instructions on how to use the tool. Please reach out with any questions (marganey@iu.edu or maryganey188@gmail.com).

### Purpose
The purpose of this tool is to convert the .imscc format from Schoology to a format that can be uploaded to google's script tool and automatically upload google-form quiz formats to the user's google drive. Considering google's tool for converting .imscc to google formats does not properly work for all files yet, this tool is meant to be the interim filler until the tool is fixed.
